# 💍 Shiv Patel Wedding 2027 — Vercel

Pure static website. No database. No backend. Just push to GitHub and deploy on Vercel.

## 📁 Files
```
shiv-wedding/
├── public/
│   └── index.html    ← The entire website
├── vercel.json       ← Vercel routing config
└── README.md
```

## 🚀 Deploy Steps

### 1 — Push to GitHub
```bash
git init
git add .
git commit -m "Shiv wedding website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/shiv-wedding.git
git push -u origin main
```

### 2 — Deploy on Vercel
1. Go to vercel.com → New Project → Import your GitHub repo
2. Framework: **Other**
3. Root Directory: leave blank
4. Click **Deploy** — no env vars needed!

### Done ✅
Your site is live at `https://shiv-wedding.vercel.app`
